----------------
## ゲーム概要
----------------
SLP (Slide Looper Puzzle) は、
ブロック単位のスライド操作によって
ターゲットパターンを再構成するロジックパズルです。

すべての問題は、正当な操作のみで生成されており、
必ず解くことが可能です。

-------------------------
## Version2 追加機能 （特別モード）
-------------------------
お使いのデバイス（PC/スマートフォン）内にある任意の画像や動画を盤面に設定し、パズルとしてプレイすることが可能です。
※端末のスペックにより、高解像度の動画等は動作が重くなる場合があります。ご自身の環境に合わせてお楽しみください。

----------------
## 遊び方
----------------
ドラッグ / スワイプ
・通常：行または列をブロック単位でスライドします。
・Shiftキー併用 / 長押しドラッグ：枠単位（サブグリッド単位）での一括移動。
・Ctrlキー併用：1マス単位の微調整（解析用・PCのみ）。

クリア条件
・画面右（または下）の「TARGET VIEW」と同じ配置に揃えるとクリアです。

-------------------------
## ゲーム演出
-------------------------
画面回転ギミック
・外周ゲージが一周すると盤面が90°回転します。位置関係の把握能力が試されます。

スポットライトギミック（記憶力チャレンジ）
・盤面に覗き穴フィルターをかけます。ボタンを押すごとにサイズが S / M / L / OFF と切り替わります。

同色フラッシュギミック
・盤面が混迷していても、同じ枠（グループ）に属するピースを一斉に発光させて視認性を高めます。
